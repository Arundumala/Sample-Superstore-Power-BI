# sample-superstore-Power-Bi
Power BI Sample Superstore Sales Dashboard

This repository contains a Power BI dashboard analyzing the Sample Superstore sales data. The dashboard provides insights into sales performance, customer behavior, and regional trends.

Repository Contents

SampleSuperstoreDashboard.pbix – Power BI file containing the dashboard.

SuperstoreSalesData.csv – Dataset used for the analysis.

Screenshots/ – Folder containing images of the dashboard visuals.

Project Overview

This project focuses on analyzing sales data for a specific region, including metrics such as total sales, quantity sold, profit and loss, sales by category and sub-category, regional and segment-wise performance, and top-performing products. The dashboard helps stakeholders understand sales trends, high-performing products, and customer segmentation for better decision-making.

Dashboard Features

Sales KPIs: Visuals showing total sales, profit, and quantity.

Category Insights: Breakdown of sales and profits by product categories.

Regional Analysis: Focused view of sales performance by a selected region.

Top Products: Visual showing top-selling products in the selected region.

Interactive Filters: Users can filter data by region, category, or segment to explore insights dynamically.

Tools Used

Power BI Desktop – Dashboard creation and visualization.

CSV Dataset – Superstore sales data used for analysis.

Insights

Focused analysis helps identify key sales drivers in the selected region.
